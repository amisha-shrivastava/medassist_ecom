JAVA SCRIPT String Functions
=============================

charAt(index)
=============
return character according to given index

Example
 <script>
     function call() {
         var x="Gwalior"
         //alert(x.charAt(2))
         var count=0
         for(i=0;i<x.length;i++)
         { var c=x.charAt(i)
             if(c=='A' || c=='a' || c=='e')
             {count++}
         }
          alert(count)
     }
 </script>
 <html>
 
 <body>
  <input type="button" value="Click Here" onclick="call()">
 </body>
</html>

=================

indexOf(str,[si])
return the index of specified str

return -1 if str does not exist

<script>
    function call() {
      var x="this is test program."

      alert(i)
      i=x.indexOf('is',i+1)
      alert(i)
      i=x.indexOf('is',i+1)

    }
</script>
